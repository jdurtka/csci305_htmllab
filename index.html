Text Editor: Sublime

From previous experience, I knew I had no interest in either emacs or vim. I have used Notepad++ in the past, and found it quite adequate for most of my needs (primarily, a basic text editor which can handle either type of line ending), but for the sake of self-improvement I opted to look at one of the editors listed in the lab. Atom didn't seem very appealing; sublime did.

I was able to locate a tutorial that looked promising, starting here. This is actually a tutorial in four parts, and delves into a number of the excellent features this editor offers, devoting the entire last two sections to some of the plugins available. A neat discovery. Naturally, this entire HTML document was generated using Sublime (Emmet makes some parts easier, but just having multiple cursors is incredibly useful when typing repeated <li> tags.)

The tutorial is divided as follows:

Part 1

  • Installation
  • Package Control (https://packagecontrol.io/installation)
  • Keyboard shortcuts (@https://scotch.io/bar-talk/sublime-text-keyboard-shortcuts)
    • The command palette keyboard shortcut Ctrl-Shift-P
    • The file switching keyboard shortcut Ctrl-P
    • Go to keyboard shortcut Ctrl-R
    • Search a file, then page Ctrl-P, @
    • Search an entire project Ctrl-Shift-R
    • Multi select and edit Ctrl-D
    • Multiple cursors with Ctrl-click
  • Other useful options

Part 2

  • Themes and color schemes
  • Project Management (Alt-Ctrl-P)
  • Vintage (Vi/Vim compatibility) Mode
  • Find and Replace (Ctrl-Shift-F)
    • Regular expression search
  • Sorting and Permuting Lists
  • Minimap
  • Configuration (default and user)

Part 3

  • Plugins
    • Advanced New File (Ctrl-Alt-N)
    • SideBarEnhancements
    • Alignment (Ctrl-Alt-A)
    • AlignTab
    • BracketHighlighter
    • ClipboardHistory (Ctrl-Alt-V)
    • ColorPicker
    • Minifier
    • HTMLPrettify
    • Snippets

Part 4

  • More plugins
    • Less2Css
    • SassforSublime
    • Emmet
    • Github Gists as Snippets
    • Git
    • Git Gutter
    • DocBlockr
    • Comment Snippets
    • Markdown Preview
    • Other useful plugins
